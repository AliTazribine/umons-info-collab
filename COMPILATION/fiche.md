# Compilation

## Description

Introduction à la compilation :
- analyse lexicale,
- analyse syntaxique descendante et ascendante,
- analyse sémantique.

Projet de compilation par groupe de deux ou trois étudiants.

## Exemple d'horaire

Q2 :
> 2h par semaine, toutes les semaines
